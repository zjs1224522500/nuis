# NetUnion Information System Rebuild 2017
## 项目简介
* NUIS2017（ NetUnion Information System Rebuild 2017），网络管理委员会内部信息系统
* 技术类型
	* 服务端基于 Java 语言，集成 Spring、Spring MVC、MyBatis 框架
	* 展示层采用HTML、CSS、JavaScript 并集成第三方库( 如 Materialize )
* 项目特色
	* 采用  Maven  进行自动化构建
	* 集成统一认证中心
*  项目开发环境要求
	* Apache Tomcat
	* MySQL
	* Java
	* Git
	* Maven
